Script simples para rolagem de vários tipos de dados, sem GUI. 
Esse script é muito simples, ele utiliza somente a biblioteca random. 
Ele pode ser usado para rolagem de dados de RPG de mesa, na falta dos dados físicos. 
Além disso, utiliza funções com o laço for e uma lista cada, o que a função faz é jogar o valor de cada rolagem dentro da lista, depois apresentar os resultados de cada rolagem mostrando a lista em si, e a soma dos valores dentro da lista para o usuário. 
Ele tem um menu bem intuitivo e instrutivo, fácil de usar.
